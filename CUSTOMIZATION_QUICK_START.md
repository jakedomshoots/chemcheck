# Report Customization - Quick Start

## 30-Second Overview

You can now pick and choose exactly what each customer sees on their service reports.

## How to Use (3 Steps)

### Step 1: Open Settings
Go to customer detail page → Click "Report Settings" button

### Step 2: Toggle Sections
Check/uncheck what you want to show:
- ☑ Overall Pool Status
- ☑ Chemical Readings
- ☑ Before/After Photos
- ☑ Service Notes
- ☑ Technician Name
- ☑ Service Duration

### Step 3: Save
Click "Save Settings" → Done!

## What You Can Control

| Item | Show/Hide |
|------|-----------|
| Overall pool status (All Good / Needs Attention) | ✓ |
| Chemical readings (pH, Chlorine, etc.) | ✓ |
| Before/after photos | ✓ |

4. **Review periodically** - Update if needs change

## Troubleshooting

### Settings not saving?
- Check internet connection
- Try again
- Contact support if issue persists

### Settings not applied to report?
- Refresh page
- Send a new report
- Verify settings were saved

### Can't find settings button?
- Make sure you're on customer detail page
- Look for gear icon (⚙️) or "Report Settings"
- Scroll down if needed

## FAQ

**Q: Can I change settings after sending a report?**
A: Yes, new settings apply to future reports.

**Q: What if I don't set preferences?**
A: All sections are shown by default.

**Q: Can customers change their own settings?**
A: No, only you can customize.

**Q: Do settings apply to SMS and Email?**
A: Yes, both use the same customized report.

**Q: Can I set different settings for different reports from same customer?**
A: No, settings apply to all reports for that customer.

**Q: What happens if I delete a customer?**
A: Settings are deleted with the customer.

**Q: Can I bulk update settings for multiple customers?**
A: Not yet, but you can update each customer individually.
