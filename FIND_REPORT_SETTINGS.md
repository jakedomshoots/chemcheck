# Where to Find Report Settings

## Location

The "Report Settings" button is on the **Customer Detail Page** in the top right corner of the customer info card.

## How to Access

### Step 1: Go to Customers
Click "Clients" in the left sidebar

### Step 2: Select a Customer
Click on any customer to open their detail page

### Step 3: Find the Button
Look at the top right of the customer info card. You'll see three buttons:

```
┌─────────────────────────────────────────────────────┐
│ Customer Name                                       │
│ Address                                             │
│                                                     │
│                    [Analysis] [Report Settings] [Log]
│                                                     │
└─────────────────────────────────────────────────────┘
```

### Step 4: Click "Report Settings"
The button has a gear icon (⚙️) and says "Report Settings"

## What You'll See

When you click the button, a dialog opens:

```
┌─────────────────────────────────────────────────────┐
│ ⚙️  Report Settings                                 │
├─────────────────────────────────────────────────────┤
│                                                     │
│ Customize what [Customer Name] sees on their       │
│ reports                                             │
│                                                     │
│ ℹ️  These settings apply to all reports sent to    │
│    this customer.                                  │
│                                                     │
│ ☑ Overall Pool Status                              │
│ ☑ Chemical Readings                                │
│ ☑ Before/After Photos                              │
│ ☑ Service Notes                                    │
│ ☑ Technician Name                                  │
│ ☑ Service Duration                                 │
│                                                     │
│ [Reset Default] [Show All] [Hide All]              │
│                                                     │
│              [Cancel]  [Save Settings]             │
└─────────────────────────────────────────────────────┘
```

## Button Location Details

### Desktop View
The button is in the top right corner of the customer card, between "Analysis" and "Log" buttons.

### Mobile View
The buttons stack vertically. "Report Settings" appears between "Analysis" and "Log".

## If You Don't See It

1. **Make sure you're on a customer detail page**
   - Click "Clients" in sidebar
   - Click on a customer name

2. **Scroll up if needed**
   - The button is at the top of the page

3. **Check browser console for errors**
   - Press F12 to open developer tools
   - Look for any red error messages

4. **Refresh the page**
   - Press Ctrl+R (or Cmd+R on Mac)

## Button Appearance

The button looks like this:

```
┌──────────────────────────┐
│ ⚙️  Report Settings      │
└──────────────────────────┘
```

- **Icon**: Gear symbol (⚙️)
- **Text**: "Report Settings"
- **Color**: Gray outline button
- **Hover**: Light gray background

## Keyboard Shortcut

Currently no keyboard shortcut, but you can:
1. Tab to the button
2. Press Enter to open

## Troubleshooting

### Button Not Visible
- Ensure you're on a customer detail page
- Try refreshing the page
- Check that JavaScript is enabled

### Dialog Won't Open
- Check browser console for errors
- Try a different browser
- Clear browser cache

### Settings Won't Save
- Check internet connection
- Verify customer has a name
- Check browser console for errors
